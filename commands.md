# Commands

## Key 
| Symbol      | Meaning                        |
| ----------- | ------------------------------ |
| (Argument)  | Argument is not required.      |

## Info
| Commands | Arguments | Description   |
| -------- | --------- | ------------- |
| 봇정보      |           | 봇의 정보를 보여줍니다. |
| 서버정보     |           | 서버정보를 보여줍니다.  |
| 유저정보     | (User)    | 유저정보를 보여줍니다.  |

## Moderation
| Commands | Arguments  | Description    |
| -------- | ---------- | -------------- |
| 밴        | User, Text | 사용자를 밴합니다.     |
| 언밴       | Text       | 사용자의 밴을 풀어줍니다. |
| 킥        | User, Text | 사용자를 추방합니다.    |

## Utility
| Commands | Arguments | Description          |
| -------- | --------- | -------------------- |
| Help     | (Command) | Display a help menu. |
| 핑        |           | 레이턴시를 보여줍니다          |
